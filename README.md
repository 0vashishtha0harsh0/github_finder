# GitHub Profile Finder 🔍

Search any GitHub username and instantly view their profile.

## Features
- Real-time GitHub API data
- Profile avatar, bio, followers, repos, following
- Direct link to GitHub profile
- Clean dark theme UI

## Tech Used
- HTML, CSS, Vanilla JavaScript
- GitHub REST API (Fetch API)

## How to Use
1. Open index.html in browser
2. Type any GitHub username
3. Click Search
